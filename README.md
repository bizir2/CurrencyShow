Необходимо реализовать сервис со следующим функционалом на языке **C#** с использованием **.NET 8**.

---

## Требования

### База данных (PostgreSQL)

В базе данных должны быть следующие таблицы:

### 1. `currency`
- `id` — первичный ключ  
- `name` — название валюты  
- `rate` — курс валюты к рублю  

### 2. `user`
- `id` — первичный ключ  
- `name` — имя пользователя  
- `password` — пароль  

Дополнительно:  
Пользователь интересуется только определённым набором валют (**favorites**).

---

## Задачи

1. Реализовать микросервис миграции БД.  

2. Реализовать фоновый сервис, который обращается по адресу:  
   `http://www.cbr.ru/scripts/XML_daily.asp`  
   и полученными данными заполняет таблицу `currency`.

3. Реализовать микросервис пользователя.  
   Требуемый функционал:
   - регистрация пользователя  
   - логин  
   - логаут  

4. Реализовать микросервис финансов.  
   Требуемый функционал:
   - получение курса валют по пользователю  

   ⚠ Пункты 3 и 4 должны быть реализованы с использованием:
   - **Clean Architecture**
   - **CQRS**

5. Реализовать авторизацию с использованием **JWT**.

6. Реализовать **API Gateway** для обоих микросервисов.

7. Добавить проект с **unit-тестами** для:
   - сервиса пользователя  
   - сервиса финансов  

   (Для фонового сервиса unit-тесты не требуются.)
